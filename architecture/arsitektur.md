 # Deskripsi Sistem Akademik Kampus
 
1. Nama Sistem
Sistem Akademik Kampus (SIK)

2. Tujuan
Sistem ini bertujuan untuk membantu proses administrasi akademik di lingkungan kampus secara digital. Dengan adanya sistem ini, kegiatan seperti pengisian KRS, penginputan nilai, dan pengelolaan jadwal kuliah dapat dilakukan dengan lebih efisien, akurat, dan transparan.

3. Pengguna Utama
Mahasiswa → Mengisi KRS, melihat jadwal kuliah, dan memantau nilai serta transkrip akademik.

Dosen → Menginput nilai mahasiswa dan melihat daftar peserta kuliah.

Admin Kampus → Mengelola data mahasiswa, dosen, mata kuliah, serta membuat laporan akademik.

4. Fitur Utama
🔐 Login & Autentikasi Pengguna  
Setiap pengguna memiliki akun dengan hak akses berbeda (mahasiswa, dosen, admin).

📚 Manajemen Data Akademik  
Meliputi data mahasiswa, dosen, mata kuliah, dan jadwal kuliah.

📝 KRS (Kartu Rencana Studi)  
Mahasiswa dapat memilih mata kuliah sesuai semester dan sistem melakukan validasi otomatis.

🧮 Input Nilai & Transkrip  
Dosen menginput nilai, sistem menghasilkan transkrip akademik secara otomatis.

📅 Jadwal Kuliah  
Admin mengatur jadwal kuliah dan mahasiswa dapat melihat jadwal sesuai kelas.

📊 Laporan Akademik  
Sistem menyediakan laporan seperti jumlah mahasiswa aktif, IPK rata-rata, dan status akademik
